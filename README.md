## Microservices Overview

Common services:
- Service Registry via Eureka server
- Config Server via Spring Cloud Config Server
- API Gateway via Spring Cloud Gateway
- Distributed Tracing via Zipkin

Business services:
- Department service
- Employee service


## System Requirment

- Spring Boot 3.0.6
- Java 17
- Maven
